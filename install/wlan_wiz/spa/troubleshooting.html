2. Durante la búsqueda de redes inalámbricas, el Asistente no ha localizado ninguna red inalámbrica.
3. La intensidad de la señal es baja o a veces inexistente.
4. (Únicamente para usuarios que no utilizan un cable USB o LAN para configurar el dispositivo inalámbrico). He cometido un error al realizar la configuración inalámbrica, pero cuando vuelvo a ejecutar el Asistente de configuración inalámbrica para introducir la configuración correcta, el Asistente no localiza ningún dispositivo.
5. El software del Asistente de configuración inalámbrica hace referencia a un punto de acceso. Sin embargo, no estoy utilizando un punto de acceso, sino un enrutador inalámbrico o un concentrador para conectarme a mi red inalámbrica.
6. No sé qué configuración de red inalámbrica tengo que utilizar, ¿qué hago?

Descripción del problema:
1. El dispositivo inalámbrico no puede conectarse a la red inalámbrica.

Solución:
SSID
Compruebe que ha introducido la información SSID, también conocida como nombre de red, correctamente cuando configure el dispositivo inalámbrico. Recuerde también que la información SSID distingue entre mayúsculas y minúsculas. Por ejemplo, si su SSID es "Red Inalámbrica" pero introduce "red inalámbrica" cuando configure el dispositivo inalámbrico, la máquina no podrá conectarse a la red, ya que las letras R e I de “Red Inalámbrica” no son mayúsculas.

CLAVE DE PASO
Compruebe también que ha introducido la clave de paso WEP/TKIP correcta al configurar el dispositivo inalámbrico. Introduzca la clave de paso correcta asegurándose que introduce el número correcto de caracteres, dependiendo de la clave de paso utilizada originalmente al configurar la red inalámbrica. En ambos casos, si ha introducido la información correcta, recomendamos que restablezca la configuración inalámbrica a los valores de fábrica del dispositivo y vuelva a ejecutar el Asistente de configuración inalámbrica. Para obtener más información sobre el modo de restablecer la configuración inalámbrica a los valores de fábrica del dispositivo inalámbrico, consulte la documentación que se proporciona con el dispositivo.

Descripción del problema:
2. Durante la búsqueda de redes inalámbricas, el Asistente no ha localizado ninguna red inalámbrica.

Solución:
Si el dispositivo inalámbrico está fuera del alcance de la red inalámbrica, acérquelo al punto de acceso inalámbrico.

Si el dispositivo inalámbrico está dentro del alcance de la red inalámbrica, compruebe que el punto de acceso inalámbrico no está ocultando el nombre de red SSID. Si es así, debe introducirlo manualmente durante la configuración del dispositivo inalámbrico. Si no está seguro de cómo hacerlo, póngase en contacto con el administrador del sistema, el proveedor de Internet o el fabricante del punto de acceso inalámbrico.

Descripción del problema:
3. La intensidad de la señal es baja o a veces inexistente.

Solución:
El mobiliario puede afectar al entorno y a la intensidad de la señal que reciben los dispositivos inalámbricos, como paredes o suelos que se encuentran entre el punto de acceso y el dispositivo. Si tiene problemas de intensidad de señal, pruebe a recolocar el dispositivo más cerca del punto de acceso.

Descripción del problema:
4. (Únicamente para usuarios que no utilizan un cable USB o LAN para configurar el dispositivo inalámbrico). He cometido un error al realizar la configuración inalámbrica, pero cuando vuelvo a ejecutar el Asistente de configuración inalámbrica para introducir la configuración correcta, el Asistente no localiza ningún dispositivo.

Solución:
Es posible que el software del Asistente no puede detectar dispositivo inalámbrico porque la configuración inalámbrica de su equipo y del dispositivo son completamente diferentes. Pruebe a restablecer la configuración predeterminada de fábrica de la impresora/dispositivo multifunción inalámbrico y vuelva a ejecutar el asistente.

Consulte el manual de usuario para obtener más información sobre cómo restaurar la configuración inalámbrica predeterminada de fábrica del dispositivo.

Descripción del problema:
7. He configurado el dispositivo para utilizar un servidor DHCP. Sin embargo, el dispositivo no está recibiendo una dirección IP del servidor DHCP. Está utilizando la dirección IP 169.254.x.x.

Solución:
