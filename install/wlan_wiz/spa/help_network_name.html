Nombre de la red inalámbrica
Introduzca el nombre de red (SSID) con el que se asociará el dispositivo.

Si el dispositivo se está comunicando en una red Ad-hoc y no hay punto de acceso, active la opción “Ésta es una red Ad-hoc y no hay punto de acceso”. Si selecciona esta opción, también tendrá que seleccionar manualmente el canal de red que esté utilizando. Si desconoce el canal de red que está utilizando, consulte la documentación que se proporciona con su sistema y busque la información en la configuración de canales.
